# Projet

- Ce repo est un starter Next.js neutre.
- Ne pas ajouter de logique client specifique dans le starter.
- Ne pas ajouter WooCommerce, WordPress ou un CMS obligatoire sans demande explicite.
- Les textes exemples vivent dans `src/content`.
- Les composants reutilisables vivent dans `src/components`.

# Regles De Travail

- Garder les routes `src/app` fines.
- Garder les changements petits et lisibles.
- Commenter les decisions techniques ou metier non evidentes, pas le code evident.
- Valider avec `npm run lint` et `npm run build` quand les dependances sont installees.
